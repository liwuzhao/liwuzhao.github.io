第二次上传的时它显示的是第一个的名字”No such app as boiling-journey-81446”

  1. git remote rm heroku
  2. git remote -v
  3. git remote add heroku https://git.heroku.com/xxx.git

ps：https://git.heroku.com/xxx.git 替换成自己的
