div / span 的不同

div一般是用来作为组合其他HTML元素的容器
span通常是作为文本的容器

class 与 id 的不同

class 属性规定元素的类名
一个元素可以规定多个类，用空格分隔类名

id属性规定HTML元素唯一的的id
一个元素只能添加唯一的id

p 与 br 的不同

p有开始标签<p>和结束标签</p>，定义HTML文档中的一个段落
br是空的HTML元素，没有关闭标签，直接使用<br>定义换行

如何使用 table 排版

table 定义表格的主题内容
thead 定义表格的页面
tr 定义表格的行
th 定义表格的表头
td 定义表格的单元

主要难度是要把admin的条件判断存到数据库里，用rails g migration add_is_admin_to_user ,代码改成

剩下就在model/user.rb加入一个action admin? ？代表这个栏目是boolean值
唯有 user 的 is_admin 这个栏位是 true 时，这个人才会被判定为 admin

然后在controller部分用这个admin?去判断
!current_user.admin?表示当前user的is_admin栏目是false，应该redirect_to root_path
这里要再放在application_controller.rb，可以同时让更多调用这个method

在admin/jobs.controller.rb中加入before_action，禁止普通使用者进行admin的所有操作

而在views页面做判断，就直接调用user.rb下的admin?去判断就可以了

对于数据存到数据库这部分，因为Rails101的教材中也没有详细讲，所以不是很懂，在做设置薪资上下限和联系方式时，也要把薪资和联系方式这几个栏位加到数据库，然后才能input。

<元学习课>学习心得

发表于 2017-03-28

元学习课就是学习如何学习的一门课，其中新概念不少，我结合目前参加全栈营的经历，大概总结一下，认为有效学习一门技能应该是这样的

  • 找到名师，重复练习高频小套路
  • 找到成就感，建立正循环
  • 不要坚持，要上瘾

学习一项技能不应该从底层理论开始

人无法用未知解释未知

因为大脑里从来没有对该项技能的认知，从理论学起很容易就会让人感到枯燥无味，碰到难以理解的问题时不由地产生畏惧感，最后成功地从入门到放弃。

高频小套路是一个新手只要跟着做，就能得到反馈，看到效果的项目。这些套路通常是由高手总结出来的，当中应该涵盖一项技能的各种基本知识。

以做菜为例，一本食谱就是一个高频小套路，你只要跟指示一步一步做，往往就能成功地做出一道菜。

编程也是一样。跟着一个高频小套路做，很容易就能得到成果，在这过程中，你只需要了解输入什么，会输出什么。

经过重复练习，这时大脑对此已经有了一定的认识，不再需要用未知解释未知了，再尝试去搞懂当中究竟发生了什么事，就显得不那么困难了。

练习高频小套路是建立正循环的一个好办法

学习需要反馈

如果投入巨大的精力，要很长时间才能得到反馈，甚至得到负面的反馈，打击会相当的巨大。

相反地，网络游戏就能非常迅速地给予玩家反馈，让玩家获得成就感，比如每次攻击跳出来的伤害数值，比如每杀掉一个怪物的增长的经验值，自然地，游戏是容易让人喜欢的。

因为高频小套路是可以直接实作得到成果的，在练习的过程中甚至会产生我简直是个天才的错觉，成就感爆棚，学习的热情自然就点燃了。

我在学习全栈营的时候，有朋友看到满屏的代码，就会觉得我很牛逼。其实他们不知道，我也不知道这屏代码是什么东西，我也只是知道输入什么，会输出什么而已 ，但是人总是虚荣的，听到别人夸你，心里总得美一番吧。

所以说，身边朋友的赞美也是一种简单地获得成就感的方法。

要上瘾，不要坚持

坚持是一种很傻逼的行为

可以说，当要用坚持来继续做下去的时候，你打从心底是不喜欢这件事的，放弃会是顺其自然，甚至理所应当的。

大师们的坚持，其实是上瘾。

Xdite老师认为建立习惯，把学习游戏化是可以让人达到上瘾状态的。

我觉得

这些是检验你是否可以对这件事的上瘾的方法。

其实跟练习高频小套路差不多，不断地得到反馈，获得成就感，找到自我价值，如果你是热爱这件事情的话，真的就会不由自主地进入上瘾状态。

这件事情其实很重要，我们通常认为兴趣是最好的老师，但是如何检验是否对一项事物有兴趣却无从下手。

以我自己为例，我曾经也认为自己是不喜欢编程的，但是在练习Xdite老师给的高频小套路时，自己却出奇地表现出了巨大的兴趣。

这时我发现

当初的不喜欢可能只是害怕

当一项高效、实用的技能摆在面前，尝试用高频小套路去入门，就是检验自己是否有兴趣的法门。

顺带一提，学习一项技能没有那么难，根据德雷福斯模型，一项领域中50%以上的人只是达到了高级新手(Advanced Beginner)的水平，可以使用基本规则完成任务。

你看，只要用对了方法，掌握一项技能可能没有那么费劲。

综上，高频小套路简直就是入门的不二法宝。

如何获得高频小套路？花钱买吧，最好是贵的。
