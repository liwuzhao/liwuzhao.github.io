报错记录<1>

发表于 2017-03-05

1

完成第二次Rails101练习后，回看时发现创建了Group后在首页无法显示

只有在My Groups上可以显示

因为在My Groups上是可以显示的，所以我猜想group是已经成功创建了的，应该是在views上出了错，在slack上发问，发现果然是在做ch08时，修改 app/views/groups/index.html.erb时出错

<%= render :partial => "group_item", :collection => @groups, :as => :group %>
collection => @groups写成了collection => @group 少了一个s

2

尝试加分题时，My Posts页面报错，找不到原因，回到branch ch07重做了一遍，解决后更新GitHub也失败，把原来的删掉重新创建一个repositories，上传的时候还是出现错误

助教指导先执行 git remote rm origin 再上传，成功解决问题
如有后续更新应执行 git push –all origin

3

尝试额外作业 新增edit和delete两个功能，思路是在controller中加入edit，delete两个action，再添加一个view页面，具体操作参考了一位同学的lowdown分享
需要定义在 app/controller/posts_controller 里面,而不是 app/controller/account/postscontroller 里面，而对应的， edit.html.erb 这个 view 页面也应该放在 app/views/posts 里面，而且还需要定义一个update。完成后发现delete功能报错
