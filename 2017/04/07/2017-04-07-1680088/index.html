div / span 的不同

div一般是用来作为组合其他HTML元素的容器
span通常是作为文本的容器

class 与 id 的不同

class 属性规定元素的类名
一个元素可以规定多个类，用空格分隔类名

id属性规定HTML元素唯一的的id
一个元素只能添加唯一的id

p 与 br 的不同

p有开始标签<p>和结束标签</p>，定义HTML文档中的一个段落
br是空的HTML元素，没有关闭标签，直接使用<br>定义换行

如何使用 table 排版

table 定义表格的主题内容
thead 定义表格的页面
tr 定义表格的行
th 定义表格的表头
td 定义表格的单元
