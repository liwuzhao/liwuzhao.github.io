修改网址id

在调用路由方法时，Rails 默认都会用 to_param 方法来转换 ID

Rails默认的to_param是这样的，我们只要修改self.id就能修改网址的ID了

比如"#{self.id}-#{self.name}"

或者给一个乱序的ID

先给Event加一个字段，比如friendly_id

>

这里面的add_index意思是给friendly打上唯一索引，便于查找

比如

@event = Event.find_by_friendly_id!(params[:id])

如果不加 index，那么程序内部的查找方法，是从 Event 第一条数据往下找，有索引之后，可以提高查找速度。

修改下面代码

@event = Event.find_by_friendly_id!(params[:id])

这就给每个Event打上了一个乱序的ID

自定义的ID的话，只要在创建event的时候自己输入friendly_id的内容就行了

表单单选UI

add_column :events, :status, :string, :default => "draft"

这里用到的验证方法是检查资料必须是只能是某些值,例如上面指status只能是STATUS这个数组里的值
